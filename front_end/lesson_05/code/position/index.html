Position

  • static - это значение по умолчанию
  • relative - расположение относительно нормального положения, как если бы элемент был статическим. Другие элементы ведут себя, как если бы релатив оставался на своем месте, то есть они не схлопываются
  • absolute - мы располагаем элемент относительно его ближайшего позиционированного родителя (position не static) - сам элемент исключается из естественного потока (связка с релатив или другим позиционированным)
  • fixed - когда элемент позициониируется относительно view-port - окна видимости - используем два из: top right left bottom
  • sticky - липкий - сначала себе ведет как статик, а после как absolute ( nb: не забыть указать top, иначе не будет залипать )
